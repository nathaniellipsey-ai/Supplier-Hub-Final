STEP 4: OPEN YOUR BROWSER

STEP 5: EXPLORE THE DASHBOARD
==============================

Now you can:

1. SEARCH:
   - Type in the search bar (e.g., "concrete", "California", "ABC Corp")
   - Press Enter or click Search
   - Results update instantly

2. FILTER:
   - On the left panel, expand filter categories
   - Click on states, categories, ratings
   - Results filter in real-time

3. VIEW SUPPLIER DETAILS:
   - Click any supplier card
   - See company info, ratings, products, certifications
   - See contact info and stock levels

4. ADD TO FAVORITES:
   - Click the heart icon on any supplier card
   - Card shows "Favorited!" message
   - Saved to your browser

5. ADD NOTES:
   - Click the notes icon on any supplier card
   - Type your notes (pricing, contact info, etc.)
   - Saved to your browser

6. USER MENU:
   - Click "Guest" in top right
   - See dropdown menu
   - View your favorites
   - View your notes
   - Logout (logs back in as guest)


STEP 6: STOP THE SERVER (when done)
=====================================

In the terminal, press:
  Ctrl + C

Server stops and terminal is ready for new commands.


TROUBLESHOOTING:
=================

PROBLEM: "Could not open requirements file"
SOLUTION: Install dependencies
  pip install Flask Flask-CORS

PROBLEM: "Address already in use" (port 3000)
SOLUTION: Kill the existing process
  Windows: netstat -ano | findstr :3000
  Mac/Linux: lsof -i :3000
  Then kill the PID
  OR use different port: python app.py --port 8000

PROBLEM: "suppliers.json not found"
SOLUTION: Regenerate suppliers
  python expand_suppliers.py

PROBLEM: "No suppliers showing"
SOLUTION: Make sure Flask server is running and returning data
  Open in browser: http://localhost:3000/api/suppliers
  Should show JSON with supplier data

PROBLEM: "Can't reach localhost"
SOLUTION: Try these addresses
  - http://localhost:3000
  - http://127.0.0.1:3000
  - http://[your-computer-ip]:3000
  - Make sure Flask server is running (check terminal)

PROBLEM: "Favorites not saving"
SOLUTION: Enable localStorage
  - Not in private/incognito mode
  - Not in a restricted browser
  - Not with storage disabled
  - Try different browser
