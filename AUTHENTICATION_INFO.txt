WHAT HAPPENS WHEN YOU OPEN THE DASHBOARD:
===========================================

1. Browser opens the page
2. JavaScript detects no user is logged in
3. System automatically creates a guest user:
   - Email: guest@supplier-portal.local
   - Name: Guest User
   - Role: Viewer
   - Saved to browser localStorage
4. Dashboard loads
5. Calls /api/suppliers endpoint
6. Flask server returns 500 suppliers from suppliers.json
7. Dashboard displays all suppliers
8. You can search, filter, favorite, and add notes
9. All your data is saved in browser localStorage


YOUR DATA (Favorites & Notes):
==============================

‚úÖ WHERE DATA IS STORED:
   Browser's localStorage on your computer
   Not on any server
   
‚úÖ WHAT'S STORED:

‚úÖ CLEARING YOUR DATA:

IF YOU FORGET YOU'RE LOGGED IN:
=================================

Don't worry! The system handles it:

1. Page loads
2. JavaScript checks localStorage for 'currentUser'
3. If not found, creates guest user
4. Saves to localStorage
5. Page loads normally

So if you:
- Close and reopen browser ‚úÖ Still logged in as guest
- Clear browser history ‚ùå You lose guest status (but just reload)
- Use different browser ‚ùå Different guest data
- Use incognito/private mode ‚ùå Data lost after closing
