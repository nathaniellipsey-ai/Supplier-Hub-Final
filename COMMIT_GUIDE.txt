================================================================================
  ‚ö†Ô∏è  COMMON ISSUES & FIXES
================================================================================

"git: command not found"
  PROBLEM: Git not installed
  FIX:
    1. Download from https://git-scm.com/download/win
    2. Install Git
    3. Restart PowerShell
    4. Try again

"fatal: not a git repository"
  PROBLEM: Folder not initialized as Git repo
  FIX:
    1. Run: git init
    2. Run: git config user.email "your@email.com"
    3. Run: git config user.name "Your Name"
    4. Run: git remote add origin https://github.com/USER/REPO.git
    5. Continue with git add, commit, push

"error: pathspec 'main' did not match any files"
  PROBLEM: No commits yet or wrong branch
  FIX:
    1. Check current branch: git branch
    2. If no commits: git add . && git commit -m "init"
    3. Try push again

"Permission denied (publickey)" or "Authentication failed"
  PROBLEM: GitHub credentials not set up
  FIX:
    1. Use HTTPS instead of SSH:
       git remote set-url origin https://github.com/USER/REPO.git
    2. Try push again
    3. Enter username and password when prompted

"Your branch is ahead of 'origin/main'"
  PROBLEM: Local commits not pushed yet
  FIX:
    1. Just run: git push origin main
    2. This will push your commits

"nothing to commit, working tree clean"
  PROBLEM: Everything already committed
  FIX:
    1. This is normal
    2. Check GitHub to verify files are there
    3. Check Render dashboard
