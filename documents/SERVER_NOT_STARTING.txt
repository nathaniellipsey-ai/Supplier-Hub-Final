WHAT npm install DOES:
=======================

package.json lists required dependencies:
  - express (web server)
  - cors (cross-origin requests)
  - ws (websockets)
  - body-parser (JSON parsing)

npm install:
  1. Reads package.json
  2. Downloads each package from npm registry
  3. Installs to node_modules/ folder
  4. Creates package-lock.json (version tracking)
  5. Takes 1-2 minutes
  6. Result: Server can now run!

IF npm install FAILS:
======================

Common errors:

  "npm: command not found"
    Problem: Node.js not in PATH
    Solution: Reinstall Node.js or add to PATH
  
  "ERR! 404 Not Found"
    Problem: Network/registry issue
    Solution: Check internet, try again later
  
  "Permission denied"
    Problem: Folder permissions
    Solution: Run as administrator
  
  "ERESOLVE unable to resolve dependency"
    Problem: Package version conflict
    Solution: npm install --legacy-peer-deps


WALMART NETWORK ISSUES:
========================

If npm install fails due to Walmart proxy:

Add proxies inline:
  HTTP_PROXY=http://sysproxy.wal-mart.com:8080 \
  HTTPS_PROXY=http://sysproxy.wal-mart.com:8080 \
  npm install

Or configure permanently:
  npm config set proxy http://sysproxy.wal-mart.com:8080
  npm config set https-proxy http://sysproxy.wal-mart.com:8080
