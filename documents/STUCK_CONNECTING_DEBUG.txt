COMMON ISSUES:
==============

ISSUE 1: Render logs show Python error
  Error example: "ModuleNotFoundError: No module named 'flask'"
  
  Cause: requirements.txt missing or incomplete
  Fix: Make sure requirements.txt has:
    Flask==2.3.3
    Flask-CORS==4.0.0
  
  Then:
    git add requirements.txt
    git commit -m "Fix requirements"
    git push
    Render will auto-redeploy

ISSUE 2: Render logs show syntax error in app.py
  Error example: "SyntaxError: invalid syntax"
  
  Cause: Python code has error
  Fix: I've created simplified app.py
  
  Make sure app.py is the NEW version
  (I rewrote it to be bulletproof)
  
  If old version still there:
    git checkout HEAD -- app.py
    git reset HEAD
    Then copy new app.py content
    git add app.py
    git push

  Response: {"success": true, "data": [], "count": 0}
  
  Cause: No suppliers generated

ISSUE 5: Dashboard says connecting forever
