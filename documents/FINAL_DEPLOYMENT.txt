APP.PY FEATURES:
==================

API Endpoints:

  GET /
    Returns: dashboard_with_api.html

  GET /api/suppliers
    Returns: All 1000 suppliers (JSON)
    Params: page=1, limit=1000

  GET /api/suppliers/:id
    Returns: Specific supplier by ID

  POST /api/suppliers/search
    Body: {"q": "search term"}
    Returns: Matching suppliers

  POST /api/suppliers/filter
    Body: {"category": "...", "region": "..."}
    Returns: Filtered suppliers

  GET /health
    Returns: Health status

TROUBLESHOOTING:
=================

If dashboard shows blank:
  1. Check browser console (F12)
  2. Look for errors
  3. Verify /api/suppliers endpoint works
  4. Check Render logs

If suppliers not loading:
  1. Check /health endpoint
  2. Should show: "suppliers_generated": 1000
  3. If not, server didn't start properly
  4. Check Render build logs

If search not working:
  1. Open browser console
  2. Check for JavaScript errors
  3. Verify /api/suppliers/search endpoint

If deploy fails:
  1. Check Render logs
  2. Verify requirements.txt has Flask and Flask-CORS
  3. Verify app.py syntax
  4. Try manual rebuild
