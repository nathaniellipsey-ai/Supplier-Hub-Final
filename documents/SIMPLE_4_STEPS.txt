================================================================================
  TROUBLESHOOTING:
================================================================================

If you see: "git: command not found"
  PROBLEM: Git not installed
  FIX: Download from https://git-scm.com/download/win, install, restart PowerShell

If you see: "fatal: not a git repository"
  PROBLEM: Not in a Git folder
  FIX: Make sure Command 1 ran successfully (you should be in the Supplier folder)

If you see: "permission denied" or "fatal: ..."
  PROBLEM: GitHub authentication issue
  FIX: When asked for password, use your GitHub Personal Access Token (not password)
       Or try: git remote set-url origin https://github.com/USER/supplier-portal.git

If nothing happens:
  FIX: Make sure you pressed Enter after each command
