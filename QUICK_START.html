Features You Can Use

Search Suppliers

Use the search bar at the top to find suppliers by name, location, or category.

Example: "concrete", "California", "ABC Corp"

Filter Suppliers

Use the left panel to filter by:

  • State (all 50 US states)
  • Category (17 construction categories)
  • Region (Northeast, Southeast, Midwest, Southwest, West)
  • Rating (3.0 to 5.0 stars)
  • Company Size (Small, Medium, Large)
  • Price Range (Budget, Standard, Premium)
View Supplier Details

Click any supplier card to see:

  • Company name & address
  • Phone number
  • Customer ratings and reviews
  • Products and services
  • Certifications
  • Lead times and response times
  • Stock levels
  • Minimum order quantities
Add to Favorites

Click the heart icon on any supplier card to add it to your favorites. These are saved in your browser!

Add Notes

Click the notes icon to add personal notes about a supplier. Great for recording contact information, pricing discussions, or other details.

Deployment Info

This dashboard is ready to deploy to Render:

  1. Push to GitHub: git push origin main
  2. Go to Render dashboard
  3. Click "Manual Deploy"
  4. Wait 3-5 minutes for deployment
  5. Access at: https://supplier-portal-2kau.onrender.com
Data: All 500 suppliers are stored in suppliers.json which is included in the deployment.

Troubleshooting

Dashboard won't load?
  • Make sure Flask server is running: python app.py
  • Check that port 3000 is available
  • Refresh your browser (Ctrl+F5)
  • Check browser console for errors (F12)
Can't find suppliers?
  • Make sure suppliers.json exists in the project folder
  • Try clearing your browser cache
  • Reload the page completely (Ctrl+F5)
Favorites not saving?
  • Check that localStorage is enabled in your browser
  • Try opening in a different browser
  • Make sure you're not in private/incognito mode (data won't persist)
